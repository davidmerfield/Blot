Bear

Write your posts with Bear

Bear is a flexible and elegant writing app for macOS and iOS designed to help you capture ideas, take notes, and write long-form content. With its powerful Markdown support, customizable themes, and organizational tools like tags, Bear is a great choice for bloggers and writers who want a simple yet powerful tool for creating content.

How to use Bear with Blot

  1. Download Bear: Get Bear from the official site, the Mac App Store, or the iOS App Store. Install and open the app.
  2. Create site on Blot: Sign up and set up your site. Decide how you want to sync your folder with Blot.
  3. Write a new post: Open Bear and create a new note. Use Markdown to structure your content with headings, lists, links, and images. Bear supports advanced Markdown features such as inline links, checkboxes, and code blocks.
  4. Export your post as Markdown: Use Bear’s export feature to save your note as a Markdown file. Go to the “More Options” menu (•••) and select Export, then choose Markdown.
  5. Publish your post: Save the exported Markdown file directly into the folder synced with Blot. Once saved, Blot will automatically publish your post to your site.

Why Use Bear?

  • Beautiful, distraction-free interface with customizable themes and typography.
  • Powerful organization tools, including tags and nested tags, to keep your notes and posts organized.
  • Markdown support for easy formatting, including inline links, code blocks, and more.
  • Seamless syncing across macOS and iOS devices using iCloud.
  • Export options for Markdown, PDF, and more to fit your workflow.

Additional Tips

  • Use Bear’s tagging system to organize drafts and published posts. Tags can be nested for better organization.
  • Leverage Bear’s Markdown formatting tips to create visually consistent and well-structured posts.
  • Enable Bear’s dark mode or choose a theme that’s comfortable for long writing sessions.
  • Bear Pro subscribers can use advanced features like cross-note links and custom export options to enhance their workflow.
