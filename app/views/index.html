Blot turns a folder into a website. Files become posts on your site. Drag-and-drop to publish.

There’s no interface, just files and folders, so you can use your favorite tools. Write your blog with Microsoft Word or Vim. Publish your photos from Lightroom. Work with others using Google Docs or git.

Blot's templates make it possible to create all sorts of sites with a few clicks. A gentle way to learn a little more about the internet. Blot's source code is dedicated to the public domain.
