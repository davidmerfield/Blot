Blot turns a folder into a website. Files inside your folder becomes posts. Drag-and-drop to publish.

There’s no interface, just files and folders, so you can use your favorite tools. Write your blog with vim or Microsoft Word. Publish your photos from Lightroom. Work on your site with others using Google Docs.

Blot is a lazy way to create a website. A gentle way to learn a little more about the web. Our templates make it possible to create a beautiful website. The source code is dedicated to the public domain.

We aim to make a tool that’s fast, dependable and helps you make the internet more beautiful.
