Blot turns a folder into a website. Files become posts on your site. Drag-and-drop to publish.

There’s no interface, just files and folders, so you can use your favorite tools. Write your blog with vim or Microsoft Word. Publish your photos from Lightroom. Work on your site with others using Google Docs.

Blot's templates make it possible to create a beautiful website with a few clicks. A gentle way to learn a little more about the internet. Blot's source code is dedicated to the public domain.

We hope you use Blot to make the internet a little more beautiful.
