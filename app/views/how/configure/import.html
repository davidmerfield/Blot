Moving to Blot

Blot can import sites from the following platforms:

  • Wordpress
  • Squarespace
  • Are.na

The importer generates a folder full of markdown files which you can drag and drop to publish immediately.

Transferring your domain

You can switch an existing domain to Blot using the DNS records documented on the domain page.

Exporting your site

You can also export your site to the Wordpress format, which is generally useful for migrating a site to another platform.
