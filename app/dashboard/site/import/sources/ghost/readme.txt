1. Install the dependencies needed for this importer:
- Run 'npm install' in the root directory of Blot's repository
- Run 'npm install' in ```app/dashboard/importer```
- Run 'npm install' in ```app/dashboard/importer/sources/ghost```

2. Retrieve the Ghost export file and then run the script on it:

node index.js path-to-export-file path-to-output-directory
