Use your own domain

Let's say you own the domain example.com and would like to use it for your blog. First, decide whether you would like to use a subdomain like www.example.com or blog.example.com, or the apex domain, which is just example.com.

How to use an apex domain

  1. Go to your domain's DNS page. This is usually where you bought the domain. Create an ALIAS record which points to blot.im.

  2. Tell Blot your domain on the dashboard.

If you cannot create an ALIAS record, I recommend following the instructions below to use the www subdomain, then set up a redirect from the apex domain to the subdomain.

How to use a subdomain

  1. Go to your domain's DNS page. This is usually where you bought the domain. Create a CNAME record for the subdomain you'd like to use and set its value to blot.im.

  2. Tell Blot your full domain, including the subdomain, on the dashboard.
