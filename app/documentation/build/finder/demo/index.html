No-Screenshot is a pure-CSS* replica of MacOS's Finder + a pair of text-editors. It's useful for software documentation and tutorials.

The library encodes some graphics using base64, but all you have to do is include a single CSS file, weighing 115kb gzip'd, less than a single screenshot. It's available as an extension to your Markdown parser, Express.js middleware, an HTML pre-processor, or client-side script.

yarn add no-screenshot

Get the code on Github

Why does this exist when you can just take a screenshot?

* More accessible than a screenshot
* Loads faster than a screenshot
* Easier to edit than a screenshot
