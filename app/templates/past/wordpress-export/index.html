Wordpress export

The purpose of this template is to generate a Wordpress export file for your blog. Since support for a Wordpress export file is so common, this template will allow you to transfer your blog to a wide variety of other platforms, including Squarespace, Micro.blog and many static site generators.

Features and known issues:

  • Export includes all public posts
  • Tags are exported for each post as Wordpress categories
  • Pages are not exported
  • Drafts are not exported
