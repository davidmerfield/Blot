Prevent a file becoming a blog post

Files and folders whose name starts with an underscore are ignored. Ignored files don’t become blog posts themselves but they can be embedded in other posts.

Keeping files private

All the files in your blog's folder are accessible to the public. If you add _post.txt to your blog's folder, someone who knows the file's path could access it. However, the only way someone could discover that file's path would be if you told them: it's not possible for vistors to your site to determine the contents of a folder.

If you want to keep _post.txt more hidden than it already is, you could add a string of random characters to its filename, e.g. _post_e81cbc3.txt. The more obscure you make the name of the file, the more effectively private it is: think of the file name as a password. But remember: anyone who knows the path to a file in your folder can access it.
