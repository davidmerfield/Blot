You can set up redirect rules for missing or moved pages on your blog. Your blog’s 404 log will help you discover broken links.

You can specify individual redirects or use a regular expression to match multiple URLs. Here are a few example redirect rules:

From To Explanation
/apples /oranges Will redirect the URL /apples to the URL /oranges, assuming there is no blog post or file with the URL /apples.
\/post\/(.*) /blog/$1 Will redirect URLs starting with /post to URLs starting with /blog.

Redirects are evaluated last. This means that a redirect will not resolve if it matches an existing blog post’s URL, or the path to a file in your folder.
