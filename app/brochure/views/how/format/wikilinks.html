Wikilinks

You can use wikilinks to create internal links between your posts. You can enable or disable wikilinks on the services page of your dashboard under Typography.

This is a wikilink to a file called *Other post.md* in the same directory: 

[[Other post]]

The wikilink above is equivalent to this Markdown:

[Other post](/other-post)

Why use Wikilinks?

You don't need to know the URL of the linked post to create a wikilink, just the filename. Also, some note-taking applications, like Obsidian, create and update wikilinks automatically.

How are Wikilinks resolved?

At the moment, Blot will first append the extension .md to what is between the brackets, then resolve it against the path to the file. For example, consider this folder below:

  • You could create a link in Introduction.md to Apple.md like so: [[Posts/apple]]

  • You could create a link in Apple.md to Introduction.md like so:[[../Introduction]] 

Introduction.md
Posts
  Apple.md

Changing the link text

You can optionally specify the link's text after a pipe inside the brackets: [[Other post|Link text]]

Tip for Obsidian

For those who use Obsidian to manage their notes, I recommend setting the New link format set to relative path to file.
