External services

Analytics

Blot makes it easy to set up Google Analytics, Heap and Clicky.

Comments

Blot makes it easy to set up Disqus.

Videos

Blot converts URLs to YouTube and Vimeo into videos. You can embed a YouTube video like this:

https://www.youtube.com/watch?v=cmaNByqIeeQ

You can paste the HTML for a video player directly into the post if you prefer.

Typography

Blot comes with a number of features to refine the typography of your blog. This includes true hanging punctuation and common typographic substitutions with Typeset. Blot also renders mathematics in \(\TeX\)and provides syntax highlighting for code.

  • Real hang­ing punc­tu­a­tion
  • Soft hy­phen in­ser­tion
  • Optical mar­gin align­ment
  • Small caps de­tec­tion
  • Punctuation sub­sti­tu­tion
