Lists

You can uses these lists in your templates. Please note that it is currently not possible to use the paginated list of entries, \{{#entries}} anywhere except the file entries.html

All of the lists have the following properties by default:

Property Description
\{{first}} boolean, true if the item is first item in the list, false otherwise
\{{last}} boolean, true if the item is the last item in the list, false otherwise
\{{penultimate}} boolean, true if the item is the penultimate item in the list, false otherwise
\{{position}} integer, indicating the position of the item in the list. For example, the first item in the list has a \{{position}} of 1 and the fourth has a \{{position}} of 4.

Entries

List Value
\{{#all_entries}}...\{{/all_entries}} Every post ever published
\{{#recent_entries}}...\{{/recent_entries}} 30 most recent posts on your site
\{{#archives}}...\{{/archives}} A list of years, months and the posts published on them

Tags

List Value
\{{#all_tags}}...\{{/all_tags}} Alphabetical list of all the tags used on your blog. Tags are case-insensitive and the case of the most recently used instance of a tag is displayed.
