Dates

A blog post’s publish date is the same as the file’s creation date by default. You can specify a different date in the file’s path or in its metadata:

Date: {{#date}}MMMM Do, YYYY{{/date}}

Blot supports a variety of date formats. These all produce a post with the same date:

{{#date}}MM/DD/YYYY{{/date}}   {{#date}}MMMM D, YYYY{{/date}}
{{#date}}M.D.YYYY{{/date}}     {{#date}}YYYY-MM-DD 00-00{{/date}}

You can use underscores, dashes, slashes and dots to seperate the numbers in the date.

Using the file’s path

You can specify a post’s publish date in its path. For example, all four of these posts share the same publish date:

 {{#date}}YYYY{{/date}} /    {{#date}}MM{{/date}} /    {{#date}}DD{{/date}} /    Post.txt
 {{#date}}YYYY{{/date}} /    {{#date}}MM{{/date}} /    {{#date}}DD-[Post.txt]{{/date}}
 {{#date}}YYYY{{/date}} /    {{#date}}MM-DD-[Post.txt]{{/date}}
 {{#date}}YYYY-MM-DD-00-00[-Post.txt]{{/date}}

Please note that a date specified in the file’s metadata takes precedence.

Schedule a post for publication

If you specify a date in the future, your post will be published then.
