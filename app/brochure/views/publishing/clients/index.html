Clients

Blot turns a folder into a blog. You can synchronize your blog's folder with any of the following clients. You can switch your blog seamlessly between clients at any point.

  • Dropbox
    A file hosting service with wide support
  • Git
    An open source distributed version control system

Switching client

You can switch seamlessly from one client to another at any point.

Multiple blogs

You can manage multiple blogs from the same account. Each blog is independent: you could use Dropbox with one and Git with another. Your blogs can use the same Dropbox account, or different accounts.
